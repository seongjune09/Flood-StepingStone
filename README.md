# 홍수디딤이
문제상황 :
최근 기후 변화로 인해 집중호우와 도시 침수 사고가 빈번하게 발생하고 있다. 

특히 지하 주차장이나 도심 지역(예 : 강남)에서는 차량이 순식간에 물에 잠기는 사례가 늘어나면서 운전자들이 미처 대피하지 못해 인명 피해로 이어지는 일이 많이 발생하고 있으며  

차량은 일정 수위 이상 물에 잠기면 문이 수압으로 인해 열리지 않고 전원이 차단되면 전동식 창문도 작동하지 않아 탈출이 불가능해지는 문제가 있다.


---

위의 문제를 해결하기 위해, 차량이 물에 잠기기 시작할 때 배터리가 완전히 침수되기 전에 창문을 자동으로 열어주는 시스템을 고안하게 되었으며

운전자가 침수 상황에서도 신속히 탈출할 수 있도록 도와 홍수 시 발생하는 인명 피해를 최소화하는 것이 이 아이디어의 핵심 목적이다.
